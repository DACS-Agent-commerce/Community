# Integrations

Bridges between DACS and other stacks: payment-rail adapters (x402, AP2, ERC-20/SPL, HTLC), DACS-2 verification-method providers (zkTLS, TLSNotary, W3C VC), substrate bindings, ERC-8004 connectors. In-tree submissions live in a subdirectory here; external ones are listed in the [INDEX](../INDEX.md#integrations). See [CONTRIBUTING](../CONTRIBUTING.md).
